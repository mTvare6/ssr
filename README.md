# ssr

Student Search TUI for IITK written in Rust.

# Building

```sh
$ git clone https://github.com/BurntSushi/ripgrep
$ cd ripgrep
$ cargo build --release
```

# Installing

```sh
$ cargo install --path .
```

# Usage

```sh
$ ssr
```

Use `C-h` `C-l` to navigate between input fields.
Input fields support emacs shortcuts.
